# Code of Conduct

This project adopts the **Contributor Covenant**, version 2.1, as its code of
conduct. The full text is available at:

<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

In short: we want a welcoming, harassment-free community for everyone. Be
respectful, assume good faith, and help keep discussions constructive.

## Enforcement

Instances of unacceptable behaviour may be reported privately to the project
maintainers. All reports will be reviewed and
investigated promptly and fairly, and reporters' privacy will be respected.

For the full enforcement guidelines and the definitions of acceptable and
unacceptable behaviour, see the linked Contributor Covenant text above.
